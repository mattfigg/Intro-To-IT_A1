Project Idea

Overview

The COVIDtrack System is a refinement and expansion of the Australian Governments COVIDsafe app that is currently available for iOS and Android mobile devices. COVIDtrack will expand on the current Australian Government solution by providing additional hardware devices that are dedicated and designed specifically for the COVIDtrack task.

The system will provide features that both track close contacts in a similar manner to the current app but instead using a dedicated hardware solution as well as providing a solution for user visit registration and venue capacity tracking in businesses which is currently left to business to implement themselves.

Motivation

The current COVIDsafe app has been received well it the Australian community with over 6.2 million people having downloaded the app in Australia (Bogle, 2020), Despite the relatively high numbers of downloads the app has room for improvement.

Current COVIDsafe issues:

  • App can easily be accidentally closed
  • App causing Bluetooth connectivity issues particularly with iPhones
  • No solution for digitally checking into a venue
  • No solution for people (children) without smart phones
  • No solution for situations where smartphone use may not be appropriate (sports)

One report suggests that the app only works 25% of the time (Taylor, 2020) and there have been reported issues with the app causing Bluetooth connectivity issues of essential medical devices on iPhones (Biggs, 2020). The rollout of the app rightly periodised speed of deployment and ease of implementation however with a possibility of no immediate cure for Covid-19 we should prepare longer term more reliable solutions to this challenge.

Description

The system will comprise of a mobile app & dongle for users and a tablet app for venues to use to check in patrons. Both the app and dongle can be used as a stand-alone products though the dongle will need to be registered and paired with the mobile phone app.

Once paired the dongle will provide the user with a quicker and more reliable solution then the app. It will also allow family members who do not have a mobile device to participate in the program.

Individuals who have tested positive for Covid or who should be quarantining will be flagged in the database, not allowing them to check into venues while they are contagious or should be isolating

User App Features

The primary function of the app will be to register details and to provide notifications to the user. Once installed and paired with the dongle users should only need to use the app to scan QR codes to check-in at venues should they choose not to use the dongle. If a dongle is detected the Bluetooth proximity function will be disables on the mobile device to conserve battery use and reduce possible Bluetooth conflicts with other apps on the device itself.

  • iOS and Android compatible
  • Allows users to enter their contact details such as name, address and contact number which will be verified through SMS
  • Family account allowing dependants without mobiles to use the dongle only
  • Provides notifications when dongle battery needs charging
  • Provides notifications to contact health department/contact tracers when suspected Covid contact has occurred
  • Provides notifications of test results
  • QR code reader for venue check-in
  • Bluetooth proximity use automatically switches off when not needed

User Dongle Features

The User dongle will be designer to be weather resistant, have a long battery life and be easily portable for ease of use. When charged the dongle will always be on and actively seeking other units or mobile devices with the COVIDtrack installed and active.

  • Will pair with mobile phone app
  • Can be paired with parents or carers phone app
  • NFC check-in system to allow quick, easy, private and secure venue check-in
  • Always-on Bluetooth to detect and record other users in close proximity
  • Charge indicator light to alert user when it needs charging
  • Attachable to keychain or handbag
  • Weather proof

Venue Check-In App Features

The venue check-in app is designed to be used on a tablet that can be set up on a stand to greet customers as they enter a venue. On arrival users will have the option of either check in using touchless NFC or to scan a QR code using the COVIDtrack app on their mobile device. The check in app can also be used at testing facilities to increase user processing speed and to automatically notify the database that the user should be isolating until they receive the test results.

  • Dedicated check-in app for tablets
  • NFC check-in system
  • QR code check-in for app only users that do not have a dongle
  • Always-on Bluetooth to detect and record other users in close proximity
    • (in case users have not checked in)
  • Venue limits to prevent more users checking in then the venues capacity

Tools and Technologies

The development of this project will require a code editor (Atom) to edit HTML, CSS and Java Script. A compiler (Eclipse) and relevant IDEs and plugins are required to develop Swift and Kotlin applications. Prototyping can be done using Figma, Sketch or Adobe XD and team collaboration with Microsoft Teams, word and Miro.
